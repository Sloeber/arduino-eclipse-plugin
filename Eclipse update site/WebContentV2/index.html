Arduino Eclipse Plugin

This website is about a free Eclipse plugin to make development of big Arduino projects easier. All you need is the Arduino IDE and this plugin. The Arduino IDE alternative is completely free and open source.

The Arduino Eclipse Plugin is a free and open source software for those who need something better than the official Arduino IDE, because especially for big projects with a lot of Codes the official IDE is really difficult to use. With the Arduino Eclipse plugin you get access to all the advanced development tools a good IDE offers, like Code Completion, Syntax Highlighting, Version Control Systems, etc. Installation is easy, all you need is the Arduino IDE (see here for supported Versions) and this plugin. Eclipse is even included in the latest Builds so you don't have to worry about downloading it separately. The Plugin supports pretty much all types of operating systems (Windows, Linux and OSX). To get started, download the latest Release including Eclipse + the Plugin and read the Installation Guide or FAQ.

Features
  • Windows, Linux and OSX support
  • 64bit ready
  • Import option for libraries + other source code
  • Integrated serial monitor + scope
  • Support for the HalfKay boot loader in Version 2
  • Simple uploading of sketches
  • Advanced build options
  • Simple install (Eclipse included in Download)
