Warning: In contrast to the arduino IDE clicking the upload button does not trigger a build.

What to do to get started quickly

Following things are needed to get started.

  • configure the plugin.
  • Set to the perspective to arduino (C/C++ is fine to).
  • Close this welcome screen.

Following things are strongly advised to do when getting started.

  • Set the save before build option.
  • add ino as a c/c++ type.

Good to know.

  • The serial monitor is in a tab in the bottom.
  • So is the scope.
  • The bottom part of the Arduino IDE showing build and upload feedback is called monitor and is in the bottom of the screen.
  • On the left you see the project explorer. This contains all the projects. Each project corresponds with a sketch.
  • There are multiple monitors in Eclipse world. If you see nothing in the monitor make sure you have the correct monitor view (buttons left top of monitor)
